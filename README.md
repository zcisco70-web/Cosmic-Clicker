# Cosmic Clicker

Juego clicker desarrollado en Unity ambientado en el espacio, donde el jugador interactúa con cuerpos celestes que evolucionan a medida que aumenta el número de clics.

Cada clic genera progreso, permitiendo desbloquear nuevas formas de planetas, estrellas y objetos cósmicos. A medida que avanzas, el universo se expande y los cuerpos celestes se transforman visualmente, ofreciendo una sensación de progresión constante.

## Juega aquí
https://ciscoz.itch.io/cosmic-clicker

## Sobre el proyecto
Cosmic Clicker es un juego incremental centrado en la evolución visual y la progresión del jugador mediante clics. El objetivo es descubrir todas las fases evolutivas de los cuerpos celestes y maximizar el progreso.

- C#
